Dependencies
------------

-  `Python > 3.8`
-  `Termcolor > 2.4`
-  `Keyboard > 0.13`

Install
-------
```bash
$ git clone https://github.com/MohNamdar/snakegame-with-python
```
```bash
$ pip install termcolor
$ pip install keyboard
```

Usage
-----

1. Run Main.py with Python Interprator :

```bash
$ python main.py
```

2. Enjoy The Game
